### Builder object

- `build` is **our** CLI command (`ng run app:build`)
- `builder` is the builder command we want to run
- `options` get passed to builder
- `configurations` optionally get passed

<CodeSurferLayout>

```json title="Sample angular.json"
{
  "custom-build": {
    "builder": "@my-org/angular-toolkit:custom-build",
    "options": {
      "browserTarget": "app:build"
    },
    "configurations": {
      "production": {
        "browserTarget": "app:build:production"
      }
    }
  }
}
```

```json title="@my-org/angular-toolkit/package.json"
{
  "name": "@my-org/angular-toolkit",
  ...

  "peerDependencies": {
    "@angular-devkit/architect": ">=0.800.0",
    "@angular-devkit/build-angular": ">=0.800.0",
    "@angular-devkit/core": ">=8.0.0",
    "@angular-devkit/schematics": ">=8.0.0"
  },

  "builders": "./builders.json",
}
```
